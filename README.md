## Mega Hack

## Desafio VTEX

Para instalar dependencias:

### `yarn install`

Para rodar um servidor localmente:

### `yarn start`

### Variáveis de ambiente

REACT_APP_API_URL = url (protocolo http) do backend
REACT_APP_URL = url (protocolo http) do frontend

## Navegando no site

### Páginas principais

Vendedor: /virtual

Loja: /shop

## Páginas do Cliente

Botão verde no canto inferior da tela: Abre uma janela que possibilita ligar para um atendente

Botão cinza no canto inferior da tela (com ícone de flecha): Abre um prompt que pede o nome do usuário e se o usuário colocar um nome ele entra para a fila de clientes à serem atendidos

Botão cinza no canto inferior da tela (com escrito 'sair da sessão x'): Usuário cancela a interação de tela com o vendedor

Botão azul no canto inferior da tela (botão do messenger): Bot utilizando o messenger como chat para sanar dúvidas rápidas.

## Página do Venderdor

Botão verde no canto inferior da tela: Abre uma janela que possibilita o atendimento de um cliente por voz

Quando Possuírem clientes na fila os nomes aparecerão em destaque na página, para iniciar uma venda basta clicar no nome.

### Comandos
tecla s: Faz scroll do lado do cliente para a posição do mouse d vendedor
tecla b: Ativa e dasitiva o 'blink' do mouse (ferramenta de destaque que segue o mouse do vendedor)
